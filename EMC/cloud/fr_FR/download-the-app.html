À usage interne d’EMC uniquement :

Pour lancer la présentation clients interactive Cloud à l’aide de l’application EMC Interactive pour iPad, suivez ces 3 étapes simples :

  1. Enregistrez votre appareil auprès d’EMC. (Pour obtenir les instructions, rendez-vous sur EMC infinIT et saisissez « Device Enrollment » dans le champ de recherche.)
  2. Accédez à l’EMC App Store.
  3. Téléchargez l’application EMC Interactive.

Notez qu’une version PowerPoint de la présentation est également disponible.
