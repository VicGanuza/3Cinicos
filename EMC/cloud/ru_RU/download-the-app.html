Информация EMC. Только для служебного пользования.

Чтобы представить интерактивную презентацию об облаке для заказчиков с помощью интерактивного приложения EMC для iPad, выполните три простых действия.

  1. Зарегистрируйте свое устройство в EMC (перейдите в EMC infinIT и выполните поиск инструкций по ключевым словам «Device Enrollment»).
  2. Получите доступ к EMC App Store.
  3. Загрузите интерактивное приложение EMC.

Обратите внимание, что в качестве альтернативы доступна также версия для PowerPoint.
