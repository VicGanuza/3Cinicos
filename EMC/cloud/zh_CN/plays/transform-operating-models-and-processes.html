实现 IT 即服务

转变运营模式和流程

  • Corporate Executive Board 基础架构执行委员会报告称，IT 领域的当务之急是“改善服务模式”
  • 自动化使 IT 服务对于企业而言更易于使用，而对于 IT 而言更易于管理

三大挑战

  1. 在创建 IT 即服务时缺少专业知识

    许多 IT 组织不知道如何利用他们的虚拟化投资来优化 IT 即服务的交付过程。他们需要有人帮助定义服务交付框架以治理和管理云服务、价格和资源调配服务并管理他们的容量。

    EMC Infrastructure Automation 服务通过提供试点项目来自动执行基础架构管理，以此加快 IT 运营向云运营模式转型。

  2. 在调配和管理服务方面需要指导

    陈旧的静态 IT 操作过程是在云和虚拟化之前开发的。针对云优化的流程若要实现自动化，需要规划服务蓝图以在其生命周期内管理服务，包括服务请求、批准、资源调配、管理、淘汰和归档。

    获得服务管理和流程编排方面的相关专业知识。定义基础架构资源调配服务，并开发服务门户以自动执行基础架构管理。

  3. 需要根据业务需求匹配 IT 服务

    IT 需要创建对业务用户而言具有重要意义的服务目录，而不是 IT 清单。这些服务应该介绍服务的成本、策略、SLA、履行流程、调配界面和所有权。这些目录应该包括 IT 提供的服务和公共云服务。

    定义服务层、构建参考体系结构并且在业务要求和服务级别之间建立关联，同时在服务背后推行技术标准。

EMC 价值

  • 加快客户从技术向服务转变的速度

    许多客户希望实现 IT 即服务，但难以定义对于业务专家有意义并支持服务自动化的服务。EMC Infrastructure Automation 服务通过定义指定服务工作流的服务模板，为使用诸如 VMware vCloud 管理套件等工具实现自动化奠定基础，帮助客户加快应用 IT 即服务。这些模板还可以为应用程序和业务利益相关方定义策略、服务层和服务级别，并为直观的自助式门户奠定基础。

  • 服务目录有助于推动标准化、转变向企业提供 IT 服务的方式，并且支持智能化地平衡成本/收益

    EMC 服务目录策略和设计服务可帮助 IT 改变向企业提供 IT 服务的方式。我们帮助 IT 使用企业了解的语言定义服务，同时采用面向消费者的 IT 服务菜单，其中指定了如何请求服务、每单位成本和服务级别。创建了注重应用程序工作负载的决策逻辑，将功能、性能和关键要求与服务级别相关联。开发参考体系结构，提供详细的技术规范并将决策逻辑转化为可行的解决方案实施方法。

  • 通过在整个生命周期内自动执行 IT 服务管理简化运营

    许多组织的 IT 流程仍然依靠手动操作和层层批准。这些流程的手动特性意味着 IT 周期过长，在与现代化公共云服务提供商相比时，这种劣势尤为明显。EMC Infrastructure Automation 服务使用服务模板帮助客户自动执行 IT 服务生命周期管理（例如，请求、批准、资源调配、管理、淘汰、归档）。EMC 全球服务的深厚专业知识与 VMware 等现代化云管理工具套件和丰富的 IT 运营经验相配合，可确保最大程度地提高运营敏捷性。

资格确认问题

  • 最高管理层级别高管、副总裁： 在您的虚拟化和云环境中，您计划如何管理 IT 服务？您的管理工具在利用云和虚拟化敏捷性方面做得怎么样？

    确定服务模式改进是否被认为是基础架构领域的第一要务。了解他们通过应用现有工具和流程是否在提高运营敏捷性方面不断取得进步，或者他们是否要使用虚拟化管理工具。确定他们当前在 IT 服务管理方面的技能水平。询问他们是否增强了对业务的响应能力。强调 EMC 深入了解 EMC/VMware 技术集成。

  • 最高管理层级别高管、副总裁： 您的团队在针对业务消费定义 IT 服务方面拥有多少经验？

    确定客户在服务自动化和服务目录定义方面的知识水平。询问他们是否具有适用于 ITaaS 的服务交付框架。强调 EMC 全球服务多年来在数据中心运营、服务管理、服务目录定义和虚拟化领域积累了丰富的经验，并且深入了解 EMC 与 VMware 技术之间的集成。

  • 最高管理层级别高管、副总裁： 您的操作过程有多么简化？为了缩短 IT 流程周期您面临多少压力？

    了解他们的 IT 组织是否难以满足业务利益相关方的周期预期，特别是在与如今的云服务提供商相比较时。强调 EMC 全球服务如何通过构建针对云优化的现代化管理环境，帮助 IT 组织简化运营。指出许多客户可以将流程周期从数周和数天缩短到几小时。

异议处理

  • “我将 EMC 视为存储供应商。你们在解决我们的运营挑战方面拥有足够丰富的经验和专业知识吗？”

    EMC 全球服务为客户部署 IT 服务管理解决方案已超过十年之久，并且针对全球客户完成了数以千计的项目部署。此外，我们拥有数千名接受了 VMware 相关培训的专家，他们了解如今虚拟化环境和云环境的管理。EMC Infrastructure Automation 服务可以帮助客户简化他们的操作过程，从而充分发挥 VMware vCloud 管理工具套件的灵活性。

  • “我想我们不需要帮助，因为我们的团队对于他们的技术领域一直管理得很好。”

    转变 IT 运营方式以实现 IT 即服务需要的不仅仅是高效管理单独的技术领域。实现 IT 即服务意味着将这些技术整合为适用于企业并且在整个解决方案生命周期内能够轻松管理的服务。EMC 全球服务可以帮助您加快运营转型，通过训练有素的专家提供 IT 服务。这些专家对如何采用对业务有意义的方式来定义服务有着深刻的理解。

  • “为什么我应该从 EMC 而不是 VMware 购买这些服务？”

    EMC 全球服务拥有丰富的基础架构专业知识，多年以来在数据中心运营方面积累了大量经验，拥有数千名接受 VMware 相关培训的专家并且与 VMware 建立了深入的合作关系，我们充分利用这些优势为全球客户提供服务，将他们的 IT 运营方式转型为实现 IT 即服务。我们与 VMware 展开合作以便充分发挥两个组织的优势， 即 EMC 在服务管理和数据中心运营领域的优势和 VMware 在虚拟化和相关管理软件方面的优势。
