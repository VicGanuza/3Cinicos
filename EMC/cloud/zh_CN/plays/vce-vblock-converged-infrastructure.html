实现 IT 即服务

VCE Vblock 聚合基础架构

  • VCE Vblock 系统是向云迁移的最快捷途径
  • Vblock 系统支持以最快的速度部署基础架构和应用程序，实现最高的应用程序性能和最低的 TCO

三大挑战

  1. IT 将 70% 的资源用在维持运营上

    随着 IT 需求不断增加，传统 IT 组织中长期存在大量思洛，创建了非常复杂且容易出错的手动流程，而组织正是使用这样的流程来部署新基础架构/应用程序以及在整个产品生命周期内维持基础架构运行。复杂的环境也带来了复杂的 IT 支持问题。

    预先设计并完全产品化的 Vblock 系统可以消除许多耗费大量人力、成本高昂并且容易出错的日常任务，这些任务会使 IT 陷入困境。

  2. IT 无法足够快速地针对业务需求做出响应

    终端用户和 LOB 巧妙地避开企业 IT，更倾向于采用内部无法提供的公共云产品。这种方法为每个应用程序的管理增加了风险、成本和复杂性， 并且违背了公司关于保护、安全性和可用性方面的标准。

    使用 VCE Vblock 系统，客户可以加快部署速度，并且极大地简化日常运营，同时提高应用程序性能和可用性。

  3. IT 需要更高效地运营

    IT 通常被企业视作成本中心，它们需要进行转型以便更好地展现 ROI。客户希望创建包含共享资源的动态池，而不是为各个应用程序配备专用硬件。他们也在想方设法衡量利用率和进行趋势分析，以便更好地规划和管理资源。

    Vblock 通过灵活、可扩展的计算、网络和存储支持混合工作负载。利用 VCE Vision Intelligent Operations 软件，客户可以从系统级别了解所有资源。

VCE 价值

  • VCE Vblock 系统是唯一真正的聚合基础架构

    Vblock 系统：

    • 由 VCE 进行工程设计，可与 EMC、Cisco 和 VMware 产品配合使用
    • 开发为以集成产品的形式进行交付
    • 通过 VCE Vision Intelligent Operations 软件进行管理
    • 作为一个产品由 VCE 提供支持
    • 通过完整记录的版本兼容性列表维持运行，以提供生命周期系统保证和完整性

    IDC 证实 VCE 客户的部署时间缩短到原来的 1/5，显著提高了可用性并节省大量 TCO。

    Gartner 最新的市场份额报告表明，VCE 在全球集成基础架构系统市场中占据了 57.4% 的份额。

  • VCE 提供集成数据保护和高可用性解决方案，可以降低风险并最大程度地提高应用程序正常运行时间

    VCE 销售、集成并支持 EMC Avamar、Data Domain 和 NetWorker 解决方案，以提供可快速并可靠实施同时降低风险的备份和恢复功能。这些解决方案包含可提高性能和效率的业界领先的重复数据消除技术。

    EMC RecoverPoint 为本地和远程站点提供 CRR 和 CDP 数据保护，并且可以紧密集成到 VMware Site Recovery Manager 中。EMC VPLEX 支持主动-主动数据中心以及直接 RecoverPoint 集成。

    VCE 数据保护功能是通过 VCE 销售并由 VCE 提供全面支持的 EMC 产品。

  • 使用 VCE Vision Intelligent Operations 可以从系统级别全面地了解 Vblock 配置、运行状况、法规遵从性和历史记录
    • 利用 VCE Vision 软件可以动态地识别系统配置和状态，使其通过开放 API 用于管理框架
    • VCE Vision 软件可以针对版本兼容性列表 (RCM) 标准规范检查配置
    • 新增功能支持高效、低风险的软件和修补程序更新以遵守最新的 RCM
    • 通过全面的日志记录功能，可以轻松地了解整个数据路径和主机的性能趋势并生成报告

竞争

竞争对手推介

  • 相较于 VCE，产品组合更大，提供更多产品（和更多选择）
  • VirtualSystem 是异构基础架构解决方案，包含多个虚拟机管理程序选项和集成管理/资源调配
  • 客户可使用 CloudSystem Matrix SW 自行安装并快速构建适用于虚拟化环境的 IaaS 云
  • 端到端高级生命周期管理
  • 单一管理环境

VCE 的回应

VCE 可将聚合基础架构集成为一个产品，而 HP 主要销售参考体系结构解决方案。

VCE 可以提供革命性的 IT 体验，能够以最快的速度部署基础架构和应用程序，同时降低风险、提高性能和可用性，并简化 IT 运营和降低 TCO。大多数 HP 解决方案无法实现真正聚合，并且不提供这些优势。

竞争对手推介

IBM PureSystems 是 IBM 对完全集成并高度灵活的新一代系统的统称。IBM 以较高折扣价将 PureFlex Systems 销售给客户，并且重点宣传平台（x86 或 POWER）、虚拟机管理程序和广泛网络和存储选项的灵活性。IBM 推介全面的服务功能来帮助客户制定他们的数据中心策略。

VCE 的回应

VCE 可将聚合基础架构集成为一个产品，而 IBM 主要销售参考体系结构解决方案。Vblock 经过了预先测试和预先设计，所采用的是大多数 IBM 系统没有使用的方式。

只有 VCE 可以提供革命性的 IT 体验，能够以最快的速度部署基础架构和应用程序，同时降低风险、提高性能和可用性，并简化 IT 运营和降低 TCO。

竞争对手推介

  • 灵活的配置（可以选择组件）
  • 使用适用于所有平台的统一存储体系结构和 ONTAP 存储操作系统提高效率
  • 存储管理与 vSphere 相集成
  • 存储资源调配和管理策略可实现 vSphere 级自动化
  • 可自定义的预先配置工作流库与 vCenter Orchestrator 相集成
  • 在任何资源级别均提供基于快照的精确数据保护

VCE 的回应

VCE 可将聚合基础架构更加紧密地集成为一个产品，而 NetApp FlexPod 是参考体系结构解决方案。FlexPod 与 EMC VSPEX 直接竞争，该产品无法提高性能、简化部署、增强可靠性或简化 IT 运营和降低 TCO，而 Vblock 系统可以提供这些优势。

资格确认问题

  • 副总裁： 您的团队是否难以满足基础架构需求，即使你们知道数据中心的大部分工作负载都过度调配，大部分资源未充分利用？

    VCE Vblock 系统可灵活配置，能够在可随时间推移进行扩展的计算、网络和存储之间提供确定性的平衡性能。这样便可确保基础架构在第一天以及产品的整个生命周期内得到最佳利用。

  • LOB： 您的 IT 团队是否能够针对日益增加的业务需求足够快速地做出响应，满足敏捷性和服务级别要求，从而为您的企业提供最佳支持？

    VCE 使 IT 可以腾出用于“维持正常运转”的时间和预算，以便更好地处理业务和提供支持。

    VCE 能够以最快的速度部署基础架构和应用程序，同时降低风险、提高性能和可用性，并简化 IT 运营和降低 TCO。通过确保系统配置出厂即可集成到目标应用程序环境中并提供可预测的性能和可用性 SLA，简化了采购/部署，从而加快价值实现速度。

  • IT： 您的 IT 团队是积极主动地发挥作用，并将精力集中在创新和推动新业务上，还是“维持正常运转”并且在跨思洛工作时遇到了困难？

    VCE Life Cycle System Assurance 可让客户摆脱管理多个供应商的修补程序和更新的复杂性。VCE 将这些修补程序集成在一起并执行回归分析，以随时间推移确保互操作性，同时 VCE Vision 软件通过将当前配置与 VCE 的版本兼容性列表进行比较以防止出现配置偏差。仅 VCE 提供该级别的生命周期系统保证，可以消除许多业务风险源。

异议处理

  • “我公司的虚拟化程度已经很高，因此未安排我购买聚合基础架构。我今年先购置存储，明年购置服务器，后年购置网络。”

    Vblock 可灵活配置，能够在可随时间推移进行扩展的计算、网络和存储之间提供确定性的平衡性能。这样便可确保基础架构在第一天以及产品的整个生命周期内得到最佳利用。

    通过确保系统配置出厂即可集成到环境中并提供可预测的性能和可用性 SLA，简化了采购/部署，从而加快价值实现速度。这样便可以降低 TCO 并提高业务敏捷性/IT 生产效率。

  • “我的员工没有时间进行更改。我们太忙了。我没有再雇用 20 个人的预算。”

    团队忙得团团转是因为 IT 太复杂。这是恶性循环，传统 IT 方法由于缺乏敏捷性和响应能力让很多企业苦不堪言。VCE 可以帮助组织实现转型，以便更好地为企业提供支持。与传统 IT 解决方案相比，规划、订购、安装 Vblock 系统、将其部署到生产环境并在整个生命周期内维持运行要简单得多。使用 VCE，客户可以让员工腾出时间从事更高效的任务，从而更好地为业务目标提供支持。

  • “这好像太复杂了，我可以从我的现有供应商那里获得，他们也有聚合基础架构。”

    VCE 拥有唯一真正的聚合基础架构。而其他解决方案无法创建代表完整基础架构的单一产品。VCE 拥有专门的工程团队来打造涵盖所有可能配置的软件。VCE 的支持组织不仅针对 Vblock 组件而且针对所有系统方面提供支持。VCE 使用正式的版本更新在整个生命周期内维持产品运行。而竞争对手做不到这一点。VCE 被客户和行业分析师评为市场份额领导者。
