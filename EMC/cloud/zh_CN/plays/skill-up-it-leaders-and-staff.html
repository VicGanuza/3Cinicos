实现 IT 即服务

提高 IT 主管和员工的技能

  • 在整个组织内培养业务主管、经理、架构师和 IT 专家等人员的云技能，以便制定 IT 战略，规划、设计 IT 并实现 IT 转型。

三大挑战

  1. 成功地实施云需要新的角色和技能

    有些组织可能认为他们团队当前的技能足以够用，不需要新的技能就可以成功地转型到云。但是，在最近的一次 EMC 调查中，有 44% 的 IT 经理表示缺少云技能是阻碍成功实施云的主要障碍。

    实施云环境需要全新的技能来解决治理、财务、组织和技术方面的问题。培训有助于加快进行成功的部署。

  2. 行业内缺少云专家

    行业内缺少精通云技术的 IT 专家，这使得很难构建合适的团队以成功迁移到云。找到具有所需云技能的人员很困难，并且成本高昂。84% 的经理希望从公司内部找到具备云技能的人才，但只有 14% 的人说他们有这样的人才。

    EMC 的云培训和认证计划旨在为 IT 专家提供成功实施云所需的技能，并提升当前 IT 团队的技术能力。

  3. 主管不了解云，还未准备迁移

    有些客户尚未准备实施云环境。业务主管不了解云，以及进行成功转型所需的步骤和流程，甚至不知道如何开始。他们不认为培训有多重要或者不将其视为该过程的促进因素。

    EMC 将为业务主管提供 90 分钟免费云和 ITaaS 课程，这些课程是了解云优势以及组织在开始着手转型时应采取哪些步骤的理想途径。

EMC 价值

  • 业务主管学习制定 IT 转型策略和计划

    许多客户想知道 IT 转型应该从何处入手以及如何开始。为什么转型如此重要？规划和执行的关键考虑因素是什么？EMC 的“用于实现业务转型的云和 IT 即服务”课程可以帮助客户规划他们的转型之旅。这些课程旨在为主管和经理提供治理、财务、组织和技术考虑因素方面的培训。简介模块以电子学习课程的形式提供，课时为一天的自学课程版本非常适合于经理和团队主管。

  • 培养云架构师的设计技能和能力

    架构师在规划和设计转型的过程中扮演着领导角色。他们需要了解策略、方向、业务目标，并设计 IT 基础架构和 IT 服务。EMC 的云架构师培训和认证计划专为起领导作用的 IT 架构师而打造。课程以云的 NIST 定义（一项行业标准）为基础。它是“开放式”课程（关注概念，而非产品培训），帮助架构师了解如何创建最适合组织的设计。架构师可以了解云概念，并学习如何设计虚拟数据中心和 IT 即服务。

  • 在 IT 组织内培养精通云技能的员工

    基于云的 IT 转型需要消除 IT 部门中的传统“思洛”。IT 部门中的每个人必须了解为什么要进行转型以及转型对企业而言有何意义。除了要精心计划内部沟通活动外，IT 领导层还必须确保所有技术人员都接受了良好的培训。EMC 的“云基础架构和服务”培训和认证计划旨在为全部 IT 技术人员提供培训和认证。此基础课程专注于概念（“开放式”），并且基于云的 NIST 定义。

竞争对手推介

不需要将钱投在员工培训或相关餐旅预算上，或者无需考虑让其团队一次离开他们的主要工作地点长达一周时间。

EMC 的回应

EMC 培训结果表明，接受培训的客户实施速度提高 34%，运营效果改善 36%，错误减少 30%，并且生产效率提高 38%。通过接受培训，客户可以规划和设计最佳的策略和体系结构。

竞争对手推介

EMC 的回应

EMC 的云课程是“开放式的”，独立于供应商，并且适用于任何供应商环境。这些课程基于美国国家标准与技术研究院 (NIST) 行业标准。EMC 还提供云认证，作为 EMC 认证专家计划的一部分，该计划是 IT 存储行业领先的培训和认证计划。

资格确认问题

  • 最高管理层级别高管： 在推动企业业务转型方面，您是否面临着内部和外部的双重压力？

    云和 ITaaS 可以推进业务创新，增加敏捷性，并提高效率，但 IT 团队需要合适的技能才能高效地工作。EMC 为组织中从业务主管到云架构师等相关人员提供云培训，以加快向云迁移的步伐。完成 EMC 云培训后，IT 经理报告称，实施速度提高 34%，运营效果改善 36%，生产效率提高 38%，并且错误减少 30%。

  • 最高管理层级别高管： 您是否正在制定包含 ITaaS 的组织转型策略？

    如果是，一个关键成功因素是拥有一个具有成功推进向云和 ITaaS 转型所需的必要技能的团队。新的技术、治理、财务和组织方法都需要全新的技能和培训。EMC 的云培训课程将有助于在组织中建立这些关键要素。高管级“用于实现业务转型的云和 ITaaS”课程从较高层面介绍了引领组织 IT 转型所需的关键要素。

  • 副总裁： 在您的 IT 组织中是否拥有合适的技能来帮助转型到基于云的 ITaaS 环境？

    最近的一次 EMC 调查表明，84% 的 IT 经理认为企业内部拥有云技能对于成功实施云非常重要，但只有 14% 的人表示他们的团队具有此类技能。具备熟练技能的团队可以更快速、更成功地完成向云和 ITaaS 的迁移，而 EMC 的云课程是为您的团队提供所需技能的绝佳方式。EMC 提供多种云课程，既有面向高管级人员的 90 分钟课程，也有面向专家级人员、课时为期一周的课程。培训是实现高效 IT 转型的关键促进因素。

异议处理

  • “为什么我的团队需要培训？”

    迁移到云和 ITaaS 模式需要 IT 组织具备与以往不同的全新技能。通过云培训，组织可以打造一个由变革推动者组成的动态团队来完成业务转型，并解决在治理、财务、组织和技术方面考虑到的问题。具备熟练技能的团队可以更好地利用技术，并提高技术投资回报率。

  • “为什么组织应该投入时间和资金为他们的团队提供培训？”

    EMC 云培训为团队提供有关如何利用云计算和 ITaaS 最大程度地推进业务创新、增加敏捷性和提高效率的见解。这些课程面向业务主管、IT 团队、IT 专业人员和专家，课时从 90 分钟到 1 周不等，并且形式多种多样，EMC 让组织可以方便地接受适合他们的培训，从而以最成功的方式实施云。

  • “是否此培训仅关注 EMC 技术或者仅在 EMC 环境中有用？

    EMC 的云培训课程以行业标准（包括 NIST 云标准）为基础，并且采用开放式形式，这表示它们不受供应商的限制并且适用于任何供应商环境。EMC 提供一系列全面的课程来加快云和 ITaaS 应用速度，既有基础级课程，也有专家和专业人员级课程。相对应的认证专家认证计划是面向技术专家的高级认证计划之一，全球颁发的证书超过 100,000 份。

传达宣传信息

EMC 的云培训旨在培养规划 IT 并将其转型到基于云的 ITaaS 所需的技能，我们为组织中包括主管、经理、架构师和 IT 专家在内的相关人员培养云技能。下面的资源可帮助您的客户从销售或技术的角度了解 IT 转型，定义业务需求以及设计基础架构和基于云的服务。
